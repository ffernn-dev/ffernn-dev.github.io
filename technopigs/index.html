Technopigs

This is a quick resource pack that I whipped up to add the feature that naming a pig "Technoblade" gives it the crown from his skin.

The idea has been suggested a lot on Twitter and elsewhere but I've made this in case Mojang doesn't end up adding it.

Optifine is required or Custom Entity Models and Entity Texture Features if you're using Fabric.

If you're using the Fabric Custom Entity Models mod, you have to make sure "Use assets/minecraft/optifine/cem" is disabled. If you haven't changed any settings you should be fine.

Thanks to u/Codepro5 for creating the bedrock version (no mods needed for beckrock)
